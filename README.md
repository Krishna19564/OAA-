# OAA+ (Online Academic Assistant) 🚀
**An AI-powered "Campus OS" for SLRTCE**

OAA+ is a unified Flutter application designed to streamline the student experience at Shree L. R. Tiwari College of Engineering. Developed for the **JARVIS 2.0 Hackathon**, it integrates academic scheduling, fee management, and smart canteen services into one seamless interface.

## ✨ Key Features
* 📅 **Academic Scheduling:** Real-time access to lecture schedules and KT tracking.
* 💳 **KT Fee Tracking:** Stay updated on pending forms and fee deadlines.
* 🍔 **Smart Canteen Ordering:** Pre-order meals to save time between lectures.
* 🤖 **Campus Chatbot:** A dedicated assistant to answer college-related queries instantly.
